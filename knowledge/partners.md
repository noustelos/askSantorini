# Partners

This file contains general partner guidance for AskSantorini.ai.

Partner data is stored only in `/data/affiliates.json`.

Important rules:
- Do not invent claims.
- Do not say "best", "official", "verified" or "top-rated" unless backed by real data.
- Use only the frontend-selected affiliate context from `/data/affiliates.json`.
- Answer the visitor's question first, then recommend if useful.
- Keep recommendations subtle and optional, never as advertisements.
